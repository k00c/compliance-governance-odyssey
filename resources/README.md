# Resources

This directory contains reference materials, templates, and useful links to support your learning journey.

## Structure

- **templates/** - Reusable templates for risk registers, audit plans, checklists, etc.
- **references/** - Saved articles, guides, and reference materials
- **reading-materials/** - Course notes, textbook summaries, and study guides

## Guidelines

- Organize materials by topic or week number
- Keep links to online resources in markdown files for easy access
- Save important reference documents locally when possible
- Maintain a glossary of key terms and acronyms
- Track course completion certificates and references to key standards
