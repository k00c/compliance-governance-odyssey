# Artifacts

This directory contains completed portfolio pieces and practical project deliverables from your learning journey.

## Structure

- **risk-registers/** - Risk assessment matrices, risk registers, and mitigation plans
- **compliance-checklists/** - Regulatory compliance checklists, gap analyses, and requirement registers
- **dashboards/** - KPI frameworks, dashboard designs, and reporting templates
- **case-studies/** - Investigation reports, stakeholder communications, and analysis documents
- **audit-reports/** - Internal audit plans, audit reports, and remediation roadmaps

## Guidelines

- Use descriptive filenames with dates (e.g., `2024-03-15-exploration-risk-register.md`)
- Include week number in filename for learning pathway artifacts (e.g., `week-03-gap-analysis.md`)
- Maintain professional formatting as if presenting to potential employers
- Document assumptions and scenarios for hypothetical projects
- Keep artifacts organized by type in appropriate subdirectories
