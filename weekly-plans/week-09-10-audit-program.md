# Weeks 09-10: Internal Audit Program Development (Milestone 4)

## Overview

**Focus Area:** Building Complete Internal Audit Function  
**Duration:** 2 Weeks  
**Estimated Time Commitment:** 20-25 hours per week (38-50 hours total)  
**Milestone Project:** Complete Internal Audit Program

## Learning Objectives

1. Develop risk-based audit universe for mining operations
2. Create annual audit plan with resource allocation
3. Design audit methodology and procedures
4. Build comprehensive template package
5. Execute sample audit and produce professional report
6. Complete **Milestone Project 4:** Full audit program package

## Milestone 4: Internal Audit Program

### Scenario
Gold mining company in Goldfields wants to establish internal audit capability. No existing audit function. Recently received DMIRS improvement notices. Upcoming external audit scheduled.

### Week 09: Program Development

**Part 1: Risk-Based Audit Universe** (4-5 hours)
- Identify 30+ auditable areas across operations
- Map to regulatory requirements and business risks
- Develop risk assessment methodology
- Rate and prioritize audit areas
- Create audit universe matrix

**Part 2: Annual Audit Plan** (4-5 hours)
- Develop 12-month audit schedule
- Allocate resources and estimate effort
- Coordinate with external audits
- Create audit calendar visualization
- Define success metrics

**Part 3: Audit Methodology Guide** (5-6 hours)
- Document planning phase procedures
- Define fieldwork standards
- Establish reporting requirements
- Create follow-up protocols
- Quality assurance approach

### Week 10: Templates and Sample Audit

**Part 4: Template Package** (3-4 hours)
- Audit charter template
- Audit plan template
- Working paper templates (5-6 types)
- Finding/observation template
- Final report template
- Action tracking template

**Part 5: Sample Environmental Compliance Audit** (8-10 hours)
- Scope: Environmental monitoring obligations
- Conduct mock audit with realistic findings
- Create complete work paper set
- Write professional audit report with:
  - Executive summary
  - Detailed findings (4-5)
  - Recommendations
  - Management responses

### Deliverables (All in `artifacts/audit-reports/`)
1. Audit Universe and Risk Assessment
2. Annual Audit Plan (12-month schedule)
3. Audit Methodology Guide (20-25 pages)
4. Template Package (6-8 templates)
5. Sample Environmental Compliance Audit Report

### Portfolio Value
Demonstrates capability to build entire audit function from scratch - senior-level competency. Shows risk-based thinking, program management, and systematic approach valued by mining companies establishing or enhancing compliance functions.
